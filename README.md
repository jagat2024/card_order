# 💳 Checkout Card UI

A clean and modern **Checkout / Order Summary Card UI** built using pure HTML and CSS.
This project focuses on layout design, flexbox alignment, and interactive UI elements.

---

## 🚀 Live Demo

👉 (https://jagat2024.github.io/card_order/)

---

## 📌 Features

* 🎨 Beautiful gradient background
* 📦 Centered card layout using Flexbox
* 🖼️ Responsive image handling
* 💳 Order summary section with plan details
* 🔘 Interactive "Proceed to Payment" button

  * Hover animation (scale + glow)
  * Click (active) effect
* ❌ Cancel order option with hover effect

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Transitions, Shadows)
